Detailed Description

A peer communicating in the medium.

dhyara::peer p ("ff:ff:ff:ff:ff:ff");
p.encrypt(false).channel(1);
p.addr().to_string(); // returns "ff:ff:ff:ff:ff:ff"
p.encrypt(); // returns false
p.channel(); // returns 1
