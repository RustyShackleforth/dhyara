Sends batch ICMP request to target.

ICMP requests in the batch are sent without any time gap between two consecutive requests. The duration of sleep between two consecutive batches can be controlled using sleep argument.

Parameters
targetthe target mac address
countnumber of ICMP batches to be sent (default 254)
batchnumber of ICMP requests in the one batch (default 1)
sleeptime (in ms) to sleep after each ICMP batch request (default to 15)
