Detailed Description

Link Layer Medium is responsible for sending packets and processing the received packets.

All received data should be forwarded to the link via _esp_sent_cb which wil then be processed through the installed actions.

dhyara::link medium; // A communication medium
dhyara::actions::acknowledgement acknowledgement(medium.local()); // Handler for acknowledgement packets
medium[dhyara::packets::type::acknowledgement] = acknowledgement;
// associate other actions ...
medium.start();
